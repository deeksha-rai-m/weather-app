# Weather App

A responsive weather application built using HTML, CSS, JavaScript, and OpenWeatherMap API.

## Features
- Search weather by city
- Real-time weather updates
- Displays temperature, humidity, and wind speed
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
